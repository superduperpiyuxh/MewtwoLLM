# MewtwoLLM — 40M Parameter LLM

A language model built from scratch with modern architecture:

- **RoPE** — Rotary Position Embedding
- **RMSNorm** — Root Mean Square Normalization
- **SwiGLU** — Gated Feed-Forward Network
- **GQA** — Grouped Query Attention

40M parameters. Trained on scraped web data + FineWeb-Edu.
